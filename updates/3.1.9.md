# What has been updated?

## Changelog - (Added, Fixed, Removed):

1. Added notifications related to employments when received a telegram or a parcel, including a webhook.
2. Added a bird system (optional) for those who want their players to buy trained birds and send telegrams (you can enable or disable the feature for players to also be able to read parcels and telegrams or allow only sending).
3. Added a mailman job support for telegrams, parcels, wardrobe, employees management and many more features in case a server owner is interested using those systems (The mailman system also includes doorlocks).
4. Fixed an issue related to billing system on telegrams that had the checkmark option enabled (even if text was hidden) on not listed jobs, this issue can cause a non-listed jobs to create bills.

2. Add the new database table related to birds:

```sql
CREATE TABLE IF NOT EXISTS `tp_bird_companions` (
  `id` int(11) NOT NULL AUTO_INCREMENT,
  `identifier` varchar(40) NOT NULL,
  `charidentifier` int(11) NOT NULL DEFAULT 0,
  `model` varchar(255) NOT NULL,
  `name` varchar(50) DEFAULT NULL,
  `age` int(11) DEFAULT 1440,
  `timestamp` longtext NOT NULL,
  `default` int(11) DEFAULT 0,
  PRIMARY KEY (`id`) USING BTREE,
  KEY `identifier` (`identifier`) USING BTREE
) ENGINE=InnoDB AUTO_INCREMENT=1 DEFAULT CHARSET=utf8mb4 COLLATE=utf8mb4_bin ROW_FORMAT=DYNAMIC;
```

9. In case you are interested using bird system for sending telegrams, you have to add the following option on `Config.Locations`:

```lua
    ['bird'] = { -- (!) THIS IS FOR BIRD SYSTEM, IT MUST BE IN THE CONFIG.LOCATIONS.
        Name = "Trained Bird",

        Coords = {x = 0.0, y = 0.0, z = 0.0, h = 0.0}, -- DO NOT TOUCH

        City = "Trained Bird",

        Hours = { Enabled = false, Duration = {am = 7, pm = 21} }, -- DO NOT TOUCH

        BlipData = { -- DO NOT TOUCH
            Enabled = false,
            Sprite = 0,
        },

        NPCData = { -- DO NOT TOUCH
            Enabled = false,
            Model = "",
            Coords = {x = 0.0, y = 0.0, z = 0.0, h = 0.0},
        },

        -- Set to true if you are using target eye for opening the mailbox office. 
        -- (!) Required an NPC Enabled, otherwise must be set to false.
        TargetEye = { Enabled = false, Script = '', Display = ''}, -- version 3.1.7 -- DO NOT TOUCH

        ActionDistance = 0.0, -- DO NOT TOUCH

        -- The following option disables reading telegrams and parcels (useful for trained birds).
        DisableReadingTelegrams = true, -- 3.1.9
    },
```

## How to update?

- You can easily update the modified files by redownloading from the **Cfx.re Portal** and copy-paste on the mentioned directory file paths.
- If new options have been implemented to the configuration file or edited, **DO NOT** copy-paste the new configuration file, otherwise you will get the latest configuration but you will lose all the changes you have made. The proper way updating a configuration file is by finding the new configuration options or the modified ones and do it **manually**.

*Cfx.re Portal **DOES NOT** force-update your configuration file or any files, if it did, you would always lose all the configuration changes you made or would totally break the script, all files are updated or edited manually.*
